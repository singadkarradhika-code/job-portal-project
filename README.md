# Online Job Portal – PHP & MySQL

An Online Job Portal System developed using **PHP and MySQL** for educational
and academic purposes. This project demonstrates how a basic job portal works,
including job posting, job searching, and applicant management.

## Features
- User registration and login
- Employer job posting and management
- Job search and filtering
- Job application system
- Admin panel for managing users and jobs
- Secure authentication
- MySQL database integration

## Technologies Used
- PHP
- MySQL
- HTML
- CSS3
- JavaScript
- Bootstrap

## Requirements
- PHP 7 or higher
- MySQL 5.7 or higher
- Apache or Nginx server
- Web browser (Chrome, Firefox, Edge, Brave)

## Installation
1. Download the project from the official source (codeastro.com).
2. Extract the project into your web server directory (e.g. `htdocs`).
3. Create a MySQL database.
4. Import the provided `.sql` file into the database.
5. Update database credentials in the configuration file (if required).
6. Start Apache and MySQL.
7. Open the project in your browser.
